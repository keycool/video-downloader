## 使用方法

当用户输入 `/video-downloader` 并附带视频 URL 时，执行以下步骤：

1. **首先阅读完整技能文档**：使用 `view_file` 工具读取 `.agent/skills/video-downloader/SKILL.md` 获取详细执行指南

2. **按照 SKILL.md 中的说明**执行视频下载任务

## 完整流程预览

```
URL → 获取标题 → 判断语言 → 创建文件夹 → 下载视频 
    → (可选) 添加中文字幕 → 获取转录文本
    → [有字幕] 直接 AI 摘要
    → [无字幕] faster-whisper 本地转录 → AI 摘要
    → 统一归档 → 返回结果
```

## 关键特性

- **自动回退**：当视频无在线字幕时，自动使用 faster-whisper 进行本地语音识别
- **多平台支持**：YouTube、Bilibili、Twitter/X 等 1000+ 平台
- **智能语言检测**：自动判断视频语言，非中文视频添加中文字幕
- **结构化摘要**：核心观点、关键洞察、金句提取、嘉宾信息、深度摘要

## 输出文件

每个视频会生成一个完整的归档文件夹：

```
download/<名称>-<日期>/
├── 视频.mp4              # 原始视频
├── 视频 [中文字幕].mp4    # 带字幕视频（如有）
├── summary.md            # AI 内容摘要
├── transcript.md         # 转录文本
├── transcript_raw.txt    # 原始语音转录（如有）
├── metadata.json         # 元数据
├── cover.jpg             # 封面图
└── *.srt                 # 字幕文件（如有）
```
